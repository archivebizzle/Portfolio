The project aims to enhance the efficiency, scalability, and security of data storage within IoT networks. The first layer involves a scalable blockchain that efficiently handles the increasing volume of transactions and data produced by IoT devices. The second layer, implemented for distributed storage, ensures the secure and decentralized storage of IoT data across the network.
